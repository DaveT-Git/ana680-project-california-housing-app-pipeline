California Housing Price Predictor

This model predicts the median house value (in $100,000 units) for California census block groups using a linear regression model. It is based on the following features from the 1990 California census data:
  • MedInc: Median income in the block group (in tens of thousands of dollars)
  • HouseAge: Median house age in the block group (in years)
  • AveRooms: Average number of rooms per household
  • Latitude: Latitude of the block group (geographic coordinate)
  • Longitude: Longitude of the block group (geographic coordinate)
